# Stay Inventory

A full-stack inventory management application for homestay owners.

## Tech Stack

### Frontend

- Next.js
- Tailwind CSS

### Backend

- Express.js
- MongoDB
- Mongoose

## Project Structure

```
stay-inventory/
├── frontend/
├── backend/
├── docs/
└── README.md
```

## Features

- Google Authentication
- Property Management
- Room Management
- Booking Management
- Availability Checker
